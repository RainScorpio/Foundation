> * BootStrap的Grip系统

### 项目中导入本地BootStrap框架

在 [BootStrap 官方网站](http://getbootstrap.com/) 下载BootStrap框架到本地. 

将BootStrap框架拷贝到项目中, 在 `<link>` 标签中引入 `bootstrap.css` 文件

代码提示中会看到bootstrap的相关CSS类提示.
